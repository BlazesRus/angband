Background

Angband 3.1.2v2 was released on 14th February 2010 and was a minor bugfix release for issues in 3.1.2.

Changelist

  • stop writing out the pricing log, to improve performance on flash drives
  • fix OS X fresh install bug
  • fix description of Berserker spell
  • unarmed attacks now allow the player to learn "notice on attack" things
  • fix colours in GCU port
  • fix the verb used when attacking monsters with slays/brands
  • fire at closest monster is now bound to 'Tab' in roguelike mode
